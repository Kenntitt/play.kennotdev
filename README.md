# KennMusic 🎵

Music player ala Spotify, powered by Nexray API.

## Cara Pakai

```bash
npm install
npm run dev
```

Buka http://localhost:3000

## Fitur
- 🔍 Search lagu via Spotify
- ▶ Autoplay ke lagu berikutnya
- 🌙 Dark / Light mode
- 📱 Mobile friendly
- shuffle, repeat, volume control
- Antrian lagu
